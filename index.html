Project Description


W4$ is an android application that scans receipt and stores the receipt data in local storage and receipt image in user's Google Drive account. Users can log in to the application using their E-mail or Google account credentials,take an image of their receipt and edit appropriate fields before saving the receipt content in a spreadsheet stored locally and the receipt image in the user's Google Drive account. The application will then show all of the receipts processed along with the total money spent in each receipt category. This data visualization is able to do data query for the receipts enabling user to be more aware of their expenses. The application can also handle receipts that are imported from external storage or e-mail providing user means to process and store reciept information.
